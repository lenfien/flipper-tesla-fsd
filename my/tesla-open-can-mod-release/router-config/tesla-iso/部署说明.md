## 这是什么

一套装在路由器上的 Tesla 网络隔离方案，**防止特斯拉远程吊销已通过 CAN 总线启用的 FSD 功能**。

车机连这个路由器后：

- ✅ WiFi 验证、中国导航、在线音乐、微信通知 — 正常
- ❌ 手机 App 控车、OTA 升级、远程诊断 — 屏蔽
- ❌ 车机状态上报、Tesla 后台远程指令通道 — 屏蔽

> 激进版可以保留手机 App 控车 + Tesla 媒体，FSD 安全等级稍降。

---

## 原理 — 一句话

**把整个 `*.tesla.cn` / `*.tesla.com` / `*.teslamotors.com` / `*.tesla.services` 域名树在 DNS 层面 sinkhole 到 `127.0.0.1`，只放行少数必需的"白名单例外"。**

```
车机 ──查询── tesla.cn ──> 路由器 dnsmasq ──> 返回 127.0.0.1 (黑洞)
车机 ──查询── connman.* ──> 路由器 dnsmasq ──> 返回真实 IP (例外放行)
车机 ──查询── baidu.com ──> 路由器 dnsmasq ──> 上游 DNS 正常解析
```

特斯拉远程禁 FSD 需要两步：

1. 车机周期性上报状态 → 后端发现 CAN 改装
2. 后端通过 hermes 长连接下发"恢复初始化"命令

**两条通道都跑在 `*.tesla.cn` 这棵树底下，一刀全砍即可。** 不需要维护具体子域清单，特斯拉后端怎么改架构都不影响。

附加防护：

- IPv6 FORWARD 全封（避免走 v6 绕过）
- Web 控制台 8888 端口仅 LAN 入站
- 持续采集车机 DNS 查询历史，自动给可疑域名打标

## 两种预设模式

| 模式 | 放行域名 | 可用功能 | FSD 安全等级 |
|---|---|---|---|
| 🛡 **保守版** | 3 条系统例外 | WiFi、中国导航、在线音乐、微信通知 | **最高** |
| ⚡ **激进版** | 保守版 + 4 条 | 上面全部 + 手机 App 控车、Tesla 媒体、语音助手 | 中 |

Web 控制台一键切换，切换后弹窗提示重启车机。

## 硬件

**任意 OpenWrt / LEDE 路由器即可。**

只要有：`dnsmasq` + `iptables` + `ip6tables` + `uhttpd` + `tar` + 基础 `sed/awk/grep`。

实测设备：Hongdian TR341（4G 工业路由器）。其它常见 OpenWrt 路由器（小米、红米、网件、Linksys 刷 OpenWrt 等）理论同样适用。

## 一键部署（推荐）

**前置条件**：电脑用网线连路由器，路由器开放 telnet（默认 admin/admin）。

```sh
cd router-config/tesla-iso
sh deploy.sh
```

脚本自动完成 9 步：

1. LuCI 登录
2. 配 WiFi（SSID 和密码通过参数指定）
3. 启 DHCP server
4. 等服务重启
5. 起本地 HTTP 服务器
6. telnet 进路由器拉包 + 跑 `install.sh`
7. 验证 Web 控制台
8. 切换到保守版
9. 跑 `verify.sh` 检测 17 个关键域名是否按预期屏蔽 / 放行

自定义参数：

```sh
sh deploy.sh --router-ip 192.168.2.254 \
             --user admin --pass YOUR_ROUTER_PASSWORD \
             --wifi-ssid YOUR_WIFI_SSID --wifi-pass YOUR_WIFI_PASSWORD \
             --pkg dist/tesla-iso-1.0.3.tar.gz
```

## 手动部署

把 `tesla-iso-1.0.3.tar.gz` 上传到路由器 `/tmp/`：

```sh
cd /tmp
tar -xzf tesla-iso-1.0.3.tar.gz
cd tesla-iso-1.0.3
sh install.sh
```

完成后浏览器开 `http://<路由器IP>:8888`。

## 给车机的最后一步

1. 车机连你部署时设置的 WiFi（SSID / 密码）
2. **禁用车机 4G 数据**（设置 → 网络）
3. **重启车机一次** — 按住方向盘左右两个滚轮 10 秒
   - 这一步**必须做**，清掉车机内的旧 DNS 缓存和 hermes 长连接

## 日常使用

浏览器开 `http://192.168.2.254:8888`：

- **运行模式**：保守版 / 激进版 一键切换
- **系统状态**：dnsmasq / 设备数 / 例外数
- **Tesla 子域档案**：车机查询过的所有 tesla 子域，自动打 ⚠（可疑）/ ✅（已放行）标
- **DNS 日志**：最近 30 行
- **升级 / 回滚**：上传新版 tar.gz 一键升级
